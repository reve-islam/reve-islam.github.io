# Brief du Projet

## Contexte
Site web de services spirituels proposant :
- Interprétation des rêves en Islam
- Diagnostic et accompagnement contre la sorcellerie (Sihr)
- Compilation gratuite d'invocations (duas) authentiques

## Objectifs Principaux
1. Offrir un accès rapide et intuitif aux informations essentielles
2. Proposer des services payants (Rêves: 20€, Roqya: 45€)
3. Fournir gratuitement 26 invocations complètes pour le quotidien
4. Être léger et optimisé pour GitHub Pages

## Contraintes Techniques
- Hébergement sur GitHub Pages (doit rester léger)
- HTML/CSS/JS pur (pas de frameworks lourds)
- Compatible mobile et desktop
- Contenu bilingue (Français + Arabe avec translittération)
- Accessibilité WCAG

## Livrables
- Site web monopage index.html
- 26 invocations complètes avec texte arabe, translittération, traduction et instructions
- Compteur de dhikr fonctionnel avec reset par appui long
- Système de paiement via Western Union, Remitly, Taptap Send
